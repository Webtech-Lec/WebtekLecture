Cascading Style Sheet(CSS)
•	Language used to specify the presentation aspects (layouts and formatting) of structurally marked up languages
•	Developed by Hakon Wium Lie (CHSS) and Bert Bos (SSP)
•	Versions:
o	CSS 1
o	CSS 2.1 (June 2011)
o	CSS 3
•	CSS Preprocessors, CSS Frameworks
o	SASS, Less, 950 Grid Sytem, Bootstrap, Foundation, Materialize, etc.
HTML/XHTML Styles Sheets
•	Author styles
o	External
o	Embedded styles
o	Inline styles
•	User styles
•	User agent styles (example default CSS 2.1 stylesheet for html 4)
CSS Statements
•	At-Rules
o	@charset
o	@import
o	@media
o	@font-face
o	@keyframe
o	@page
•	CCS Rule Sets (aka CSS Rules, Style Rules)
o	Consist of a selector, followed by a brace-enclosed /**/
CSS Selectors
•	Selector
o	Structure used as a condition in a CSS Rule to determine which elements in the document tree are matched by the selector and are thus targeted by the formatting
•	Selector Syntax
o	Chain of one or more sequence of simple selectors separated by combinators with one pseudo-element possibly appended to the last sequence
